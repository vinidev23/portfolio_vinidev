Coffe's

Desenvolvi um projeto de website para uma cafeteria utilizando as linguagens HTML, CSS e JavaScript, com foco em responsividade e experiência do usuário em múltiplos dispositivos. O sistema apresenta páginas de login e registro de usuários, cujas informações são armazenadas em um banco de dados MySQL, gerenciado por meio do ambiente XAMPP. A comunicação entre o front-end e o banco de dados foi realizada com Python, utilizando o microframework Flask, permitindo a integração e o controle de sessões dos usuários. No front-end, foram implementadas funcionalidades dinâmicas, como adição de produtos ao carrinho, desenvolvidas em JavaScript puro. O site é totalmente responsivo e adaptado a diferentes resoluções de tela, demonstrando habilidades tanto em desenvolvimento de interfaces quanto em integração com banco de dados e lógica de autenticação de usuários.
