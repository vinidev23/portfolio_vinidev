DollarBot

Compartilho com vocês um projeto especial que desenvolvi com muito aprendizado e dedicação: um ChatBot em Python com IA, capaz de responder perguntas e atualizar em tempo real a cotação do Dólar (USD) para Real (BRL). O projeto conta com Integração com API de câmbio em tempo real, Sistema de logging configurado para registrar, acompanhar interações do usuário e respostas da IA. E uma base sólida para evoluir ainda mais com novas funcionalidades e melhorias. Além de fortalecer meus conhecimentos em Python, esse projeto me proporcionou uma visão prática sobre: Manipulação de dados em tempo real, Estruturação de logs para controle e análise, e Aplicação de lógica de programação voltada para soluções inteligentes.
