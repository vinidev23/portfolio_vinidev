TaskFlow

Trata-se de uma aplicação web desenvolvida com Python (Flask) no backend, SQLite como banco de dados e integração com um frontend moderno. O objetivo é permitir o cadastro, acompanhamento e conclusão de tarefas de forma intuitiva e eficiente — ideal tanto para uso pessoal quanto para equipes.
